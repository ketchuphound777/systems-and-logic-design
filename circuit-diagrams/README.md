Circuit Diagrams
This folder contains my digital logic circuit diagrams created for coursework and practice. These diagrams represent my understanding of Boolean logic, gate‑level design, and system behavior using AND, OR, NOT, XOR, NAND, and NOR gates.

About This Folder
Some earlier files were lost due to storage limits and hardware changes, but the diagrams included here are the original versions that were preserved, along with a few older or partial drafts.

This folder includes a mix of:

final versions of circuit diagrams

older incomplete versions that show the development process

screenshots of diagrams when the original file format wasn’t available

Screenshots are included so the work can still be reviewed even when certain file types were not preserved.

Types of Diagrams Included
Boolean logic circuits

simplified and unsimplified circuits
